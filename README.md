### üöÄ What I'm building

**[Circla](https://www.withcircla.com)** ‚Äî an AI-powered tool that automatically finds the most viral-worthy moments in long-form videos and renders them as ready-to-post social media clips. The pipeline uses OpenAI Whisper for transcription, Claude API for intelligent segment analysis, and FFmpeg for rendering with auto-generated captions. Built end-to-end in Python with a modular, scalable architecture.
